|й–ұи®ҖйҮҸ:

йЎҢзӣ®ж•ҳиҝ°

  • йЎҢзӣ®йӣЈеәҰ: Easy
  • йЎҢзӣ®жҸҸиҝ°: зөҰе®ҡеӯ—дёІ pattern д»ҘеҸҠ еӯ—дёІ sпјҢжӘўжҹҘ s жҳҜеҗҰйҒөеҫӘзӣёеҗҢжЁЎејҸ

йЎҢзӣ®жңүиӘӘжҳҺйҖҷиЈЎ зӣёеҗҢжЁЎејҸ зҡ„ж„ҸжҖқ: pattern дёӯзҡ„д»»дёҖеӯ—е…ғиҲҮ s дёӯзҡ„йқһз©әеӯ—дёІе®Ңе…ЁеҢ№й…Қзҡ„ 1е°Қ1жҳ е°„й—ңдҝӮпјҢе…·й«”дҫҶиӘӘиҰҸеүҮеҰӮдёӢ:

  • еңЁ pattern дёӯзҡ„жҜҸеҖӢеӯ—жҜҚпјҢйғҪжңүжҳҺзўәзҡ„е°ҚжҮүеҲ° s дёӯзҡ„ unique еӯ—дёІ
  • еңЁ s дёӯзҡ„ unique еӯ—дёІпјҢжҳҺзўәе°ҚжҮүеҲ° pattern дёӯзҡ„дёҖеҖӢеӯ—жҜҚ
  • жІ’жңүе…©еҖӢеӯ—жҜҚжҳ е°„еҲ°еҗҢдёҖеҖӢе–®и©һпјҢд№ҹжІ’жңүе…©еҖӢе–®и©һжҳ е°„еҲ°еҗҢдёҖеҖӢеӯ—жҜҚ

и§Јжі•

дёҖй–Ӣе§Ӣзҡ„жғіжі•

йҖҷиЈЎжғіжі•еҫҲе–®зҙ”е°ұжҳҜйҖҸйҒҺ Hash Table дҫҶеҺ»е»әз«ӢдёҰе„Іеӯҳжҳ е°„й—ңдҝӮгҖӮиҖҢеңЁе»әз«ӢйҒҺзЁӢдёӯеҸҜд»ҘеҺ»жӘўжҹҘз•¶еүҚзҡ„еӯ—жҜҚе’Ңе°ҚжҮүеӯ—дёІжҳҜеҗҰе·Із¶“еӯҳеңЁжҳ е°„й—ңдҝӮж–ј hash table дёӯпјҢеҰӮжһңжңүж•‘еӣһеӮі falseпјҢеҰӮжһңж•ҙеҖӢ hash tableйғҪе»әз«ӢеҘҪеҫҢйғҪжІ’жңүйҮҚиӨҮзҡ„жҳ е°„й—ңдҝӮйӮЈе°ұеӣһеӮі trueгҖӮ

жҲ‘зҡ„и§Јжі•

жҲ‘еҖ‘е®Је‘ҠдәҶе…©еҖӢ unordered_map еҲҶеҲҘз”Ёж–јеӯҳж”ҫ pattern -> s д»ҘеҸҠ s -> pattern зҡ„жҳ е°„й—ңдҝӮ:

жҺҘи‘—пјҢйңҖиҰҒе°Үеӯ—дёІ s е„ІеӯҳжҲҗйҷЈеҲ—пјҢйҖҷиЈЎйҖҸйҒҺ std::istringstream дҫҶеҺ»дҫқеәҸеҲҶеүІ s дёҰеӯҳж”ҫеҲ°йҷЈеҲ— s_list дёӯпјҢ

иӢҘиҰҒдҪҝз”Ё std::istringstreamпјҢйңҖиҰҒеј•е…ҘжЁҷй ӯ #include <sstream>пјҢе®Је‘Ҡж–№ејҸеҰӮдёӢ:

д№ӢеҫҢйҖҸйҒҺ while иҝҙеңҲд»ҘеҸҠйҒӢз®—еӯҗ >> е°ұеҸҜд»Ҙе°Үеӯ—дёІе°Һеҗ‘еҲ°е®Је‘ҠдҫҶеӯҳж”ҫеӯҗеӯ—дёІзҡ„и®Ҡж•ёдёӯйҖІиЎҢиҷ•зҗҶ

жҺҘи‘—е°ұжҳҜеңЁ pattern дёӯпјҢдҫқеәҸе»әж§Ӣ hash table:

дёҖж—ҰжІ’жңүеңЁ umap дёӯжүҫеҲ°з•¶еүҚ pattern дёӯзҡ„еӯ—жҜҚ (Ex. a) еүҮе»әз«Ӣе°ҚжҮүй—ңдҝӮ

иҖҢеҰӮжһңжңүжүҫеҲ°пјҢеүҮйңҖиҰҒжӘўжҹҘд»ҘгҖҖa зӮә Key зҡ„е°ҚжҮү Value жҳҜеҗҰзӮәз•¶еүҚзҡ„ s_wordпјҢеҰӮжһңдёҚзӯүж–јпјҢйӮЈе°ұеӣһеӮі falseпјҢдҫӢеҰӮ

дёҖж—ҰжӘўжҹҘе®ҢжҲҗеҫҢпјҢд№ҹе°ұд»ЈиЎЁ pattern жҳ е°„еҲ° s зҡ„й—ңдҝӮжІ’е•ҸйЎҢгҖӮжҺҘи‘—е°ұйңҖиҰҒжӘўжҹҘ s еҲ° pattern зҡ„жҳ е°„й—ңдҝӮ

еҗҢдёҠпјҢдёҖж—ҰжІ’жңүеңЁ uamp2 дёӯжүҫеҲ°з•¶еүҚ s_word дёӯзҡ„еӯ—дёІ (Ex. dog) еүҮе»әз«Ӣе°ҚжҮүй—ңдҝӮ

иҖҢеҰӮжһңжңүжүҫеҲ°пјҢеүҮйңҖиҰҒжӘўжҹҘд»ҘгҖҖdog зӮә Key зҡ„е°ҚжҮү Value жҳҜеҗҰзӮәз•¶еүҚзҡ„ str (pattern[i])пјҢеҰӮжһңдёҚзӯүж–јпјҢйӮЈе°ұеӣһеӮі falseпјҢдҫӢеҰӮ
