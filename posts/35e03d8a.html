йҰ–е…ҲеҒҮиЁӯд»ҠеӨ©йҷЈеҲ—жҳҜ [1,2,5] йӮЈе°Қж–ј amount жҜҸж¬ЎеҸҜд»ҘйҒёж“Үжёӣ1жёӣ2жҲ–жёӣ5пјҢжүЈйҷӨе®Ңз•ўеҫҢдёӢдёҖијӘеҸҲеҸҜд»ҘйҒёж“ҮиҰҒжёӣ1жёӣ2йӮ„жҳҜжёӣ5пјҢзӣҙеҲ°жңҖеҫҢеҰӮжһң amount зӮә 0 еүҮд»ЈиЎЁйҒ”еҲ°зӣ®жЁҷйҮ‘йЎҚпјҢеҰӮжһңжүЈйҷӨеӨӘеӨҡ amount < 0 е°ұд»ЈиЎЁдёҚиғҪйҒ”еҲ°зӣ®жЁҷйҮ‘йЎҚпјҢйҖҷжҷӮеҖҷе°ұйңҖиҰҒеӣһеӮі -1гҖӮеҰӮжһңз•«еҮәйҖҷеҖӢжғіжі•зҡ„йҒһиҝҙжЁ№жңғеғҸжҳҜдёӢйқўйҖҷжЁЈгҖӮ

Step1 и§Јжі•зҡ„йҒһиҝҙжЁ№

еҸҜд»ҘзңӢеҲ°жҮүи©ІжңғжңүеҫҲеӨҡйҮҚиӨҮзҡ„иЁҲз®—пјҢеӣ жӯӨзЁҚеҫҢд№ҹжңғжңүжңҖдҪіеҢ–зҡ„з©әй–“еңЁгҖӮ

жҲ‘зҡ„и§Јзӯ”

Step-1

жӯҘй©ҹдёҖе°ұжҳҜе…Ҳжұәе®ҡеҹәжң¬зҡ„йҒһиҝҙй—ңдҝӮејҸпјҢд»ҘеҸҠ base caseгҖӮйҰ–е…ҲеҸӘиҰҒ amount жүЈеҲ°0е°ұжңғеӣһеӮі0пјҢиҖҢжүЈи¶…йҒҺзҡ„и©ұе°ұеё¶иЎЁдёҠдёҖжӯҘйҒёж“Үзҡ„йқўйЎҚжІ’иҫҰжі•йҒ”еҲ°зӣ®жЁҷйҮ‘йЎҚгҖӮз”ұж–ј coins йҷЈеҲ—зҡ„жҜҸеҖӢе…ғзҙ йғҪеҸҜд»ҘйҒёжҲ–дёҚйҒёпјҢеӣ жӯӨйҖҸйҒҺиҝҙеңҲпјҢи®“жҜҸеҖӢе…ғзҙ йғҪеҺ»з”Ё amount еҺ»жүЈжҺүз•¶еүҚе…ғзҙ пјҢйҖІе…ҘдёӢдёҖеұӨйҒһиҝҙпјҢиҖҢйҒһиҝҙиҝ”еӣһзҡ„зөҗжһңжңғиў«еӯҳж”ҫеҲ° resultпјҢд»ЈиЎЁзЎ¬е№Јж•ёйҮҸпјҢеҰӮжһңйҖҷеҖӢ result > 0 е°ұд»ЈиЎЁпјҢеҫһи©Іжўқи·Ҝеҫ‘жүҫеҲ°дәҶдёҖеҖӢеҸҜиЎҢзҡ„и§ЈпјҢдёҰдё”жҲ‘еҖ‘жӣҙж–°жңҖе°Ҹзҡ„зЎ¬е№Јж•ёйҮҸ minCoinsгҖӮ йҖҷиЈЎд№ӢжүҖд»ҘиҰҒеҠ  1пјҢжҳҜиЎЁзӨәдҪҝз”ЁдәҶз•¶еүҚзЎ¬е№ЈпјҡжҜҸз•¶жҲ‘еҖ‘еҳ—и©ҰдёҖжһҡзЎ¬е№ЈжҷӮпјҢжңғжёӣеҺ»и©ІзЎ¬е№Јзҡ„йқўеҖјпјҢдёҰйҒһиҝҙе‘јеҸ« coinChangeHelper дҫҶиЁҲз®—еү©йӨҳйҮ‘йЎҚзҡ„зЎ¬е№Јж•ёйҮҸгҖӮйҖҷжҷӮеҖҷпјҢеӣ зӮәжҲ‘еҖ‘е·Із¶“йҒёж“ҮдәҶдёҖжһҡзЎ¬е№ЈпјҢжүҖд»ҘзёҪж•ёйңҖиҰҒеҠ дёҠ 1гҖӮиҖҢеүҚйқўзҡ„жўқд»¶ minCount == -1 еүҮд»ЈиЎЁз¬¬дёҖж¬ЎжүҫеҲ°жңүж•Ҳи§ЈгҖӮ

жңҖеҫҢе°ұжҳҜиҝ”еӣһ minCount

йҖҷжЁЈеҲқжӯҘзҡ„еҒҡжі•еҰӮжһңеңЁ coins е’Ң amount ж•ёеҖје°Ҹзҡ„жҷӮеҖҷйӮ„дёҚжңғеҮәдәӢпјҢдҪҶеҰӮжһңж•ёеӯ—дёҖеӨ§пјҢе°ұжңғ Time Limit ExceededгҖӮеӣ жӯӨжңғйңҖиҰҒйҖІиЎҢз¬¬дәҢжӯҘй©ҹ - Memoization

Step-2

йҖҷиЈЎеҸҰеӨ–е®Је‘ҠдәҶдёҖеҖӢйҷЈеҲ— dp з”ЁдҫҶе„ІеӯҳйҮҚиӨҮзҡ„иЁҲз®—пјҢдҪҶйҖҷжЁЈеҒҡеҰӮжһңеҺ»еҹ·иЎҢдёҖжЁЈжңғжҳҜ Time Limit Excceded! еҺҹеӣ еңЁж–ј minCountеҲқе§ӢеҖјзҡ„иЁӯе®ҡ

з”ұж–јжҲ‘еҖ‘зҡ„ dp д№ҹйғҪеҲқе§ӢеҢ–жҲҗ -1пјҢdp[amount]=-1 д»ЈиЎЁйҖҷеҖӢйҮ‘йЎҚе°ҡжңӘиў«иЁҲз®—йҒҺпјҢиҖҢ result=-1еүҮд»ЈиЎЁеҲҶж”Ҝз„Ўжі•жүҫеҲ°жңүж•Ҳи§ЈпјҢд»ЈиЎЁзҡ„еүҚйқўйЎҚзө„еҗҲз„Ўжі•йҒ”еҲ°зӣ®жЁҷйҮ‘йЎҚпјҢз•¶ result == -1 жҷӮпјҢжңғеҝ…й ҲжӘўжҹҘ minCount == -1пјҢд»ҘзўәдҝқеңЁйҖҷдәӣз„Ўж•Ҳи§ЈдёӯжүҫеҲ°жңүж•Ҳзҡ„жңҖе°Ҹи§ЈгҖӮйҖҷжЁЈзҡ„ -1 йӮҸијҜжңғи®“жҜҸж¬ЎжӘўжҹҘ minCount жҷӮпјҢйғҪеҝ…й ҲйЎҚеӨ–иҷ•зҗҶ -1 зҡ„жўқд»¶пјҢдёҰдё”еңЁ dp зҡ„еҖје°ҡжңӘжӣҙж–°пјҲд»ҚзӮә -1пјүжҷӮпјҢд№ҹжңғйҖІиЎҢеӨҡж¬ЎйҮҚиӨҮиЁҲз®—гҖӮ

йҖҷиЈЎж”№жҲҗе°Ү minCount еҲқе§ӢеҢ–жҲҗ INT_MAXпјҢе°ұдёҚжңғеҮәзҸҫеүӣеүӣйӮЈзЁ®гҖҢжӯӨ -1 йқһеҪј -1 зҡ„зӢҖжіҒгҖҚеҸҜд»Ҙжёӣе°‘еӨҡйӨҳзҡ„еҲӨж–·гҖӮ з•¶ result == INT_MAX жҷӮпјҢиЎЁзӨәз„Ўжі•йҒ”еҲ°зӣ®жЁҷйҮ‘йЎҚпјҢжҲ‘еҖ‘еҸҜд»Ҙз°Ўе–®ең°жӘўжҹҘ result != INT_MAX дҫҶеҲӨж–·жңүж•Ҳи§ЈгҖӮеҸҰеӨ–д№ҹйҖҸйҒҺ min() дҫҶеҺ»еҲӨж–· minCount йӮ„жҳҜйҒһиҝҙе‘јеҸ«зөҗжһңе“ӘеҖӢжҜ”ијғжңғе°ҸпјҢд№ҹиғҪжёӣе°‘еҲӨж–·зҡ„иӨҮйӣңеәҰгҖӮ
